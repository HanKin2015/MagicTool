# 1、路由
查看当前路由：
route print


route add 199.200.2.2 mask 255.255.0.0 199.200.255.254 -p

加p参数会加到永久路由中

同理删除：route delete 199.200.2.2 mask 255.255.0.0 199.200.255.254 -p
