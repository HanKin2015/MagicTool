# tar.gz文件
解压：tar -zcvf 压缩文件名.tar.gz 被压缩文件名
压缩：tar -zxvf 压缩文件名.tar.gz

# gz文件

# zip文件
解压：unzip target.zip -d 文件夹名
压缩：zip 文件夹名

# tar文件
没有解压并新建目录的命bai令，只能先建立目录du，然后把文件解压到zhi这个目录下。
需要先新建文件夹，默认在当前文件夹下
解压：tar xvf filename.tar -C 文件夹名
压缩：tar cvf
